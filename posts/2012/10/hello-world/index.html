Hello world!

Mon 15 October 2012 By Xabier Larrakoetxea

This is the first post of many. I will start this blog with a classic:

def say_hello():
    print("Hello world!")

if __name__ == "__main__":
    say_hello()

This blog will have one theme in mind: Technology. If I am more specific: GNU/Linux, programming and scalability.

The blog is made with Pelican and gh-pages. The question maybe is Why I want this blog to be static? Well I will resume this in a list:

  • Simple (I like simple things: Python, Slackware...)
  • Scalable (all is static!)
  • Posts in reStructuredText and/or markdown syntax (This can be ported easily in the future)
  • Github pages FTW!
  • Pelican is awesome (Jekyll too, but I prefer python to ruby)
  • Is fun and geeky :)

I think that I made my point. See you soon with a techie and geeky post
